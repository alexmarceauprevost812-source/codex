# Vidéos

Placez votre vidéo de fond ici sous le nom `background.mp4`.

- Format recommandé : `.mp4` encodé en H.264 (compatibilité maximale).
- Pour utiliser un autre nom ou format, modifiez `app/page.tsx` :
  `<VideoStage src="/videos/votre-fichier.webm" />`.

La vidéo est jouée 4 secondes en plein écran, puis passe en fond
à 40 % d'opacité avec l'interface du chat par-dessus.
